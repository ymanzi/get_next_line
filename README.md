# Get Next Line :book:
Function reading a line on a file descriptor

## Subject :mag:

[GET NEXT LINE SUBJECT](https://cdn.intra.42.fr/pdf/pdf/13322/en.subject.pdf)

## Return Values
* -1 : An error occurs :x:
* 0 : All has been read. (EOF) :white_circle:
* 1 : A line has been read :white_check_mark:
